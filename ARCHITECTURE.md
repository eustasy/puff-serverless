# Architecture

## Actions
- Register
  - Verify Email
- Log in
  - Verify password
  - Verify 2nd factor
  - Create a session
- Log out
  - Terminate a session
- Reset password
- Account Mangement
  - Add backup email
  - Change primary email
  - Change password
  - Add 2nd factor
  - Remove 2nd factor
  - List sessions
  - Terminate all sessions
